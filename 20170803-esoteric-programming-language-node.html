Example

ex.aheui밯망희
Takes a UTF-8 character and returns UTF-8 code point value

밯 = ㅂ + ㅏ + ㅎ

ㅂ - Push to current storage

ㅏ - Move to right

ㅎ - Receive a UTF-8 character

망 = ㅁ + ㅏ + ㅇ

ㅁ - Pop from the current storage

ㅏ - Move to right

ㅇ - Noop

희 = ㅎ + ㅢ

ㅎ - Terminate.
(Pop from the current storage and return it to OS)

ㅢ - Reflect cursor
